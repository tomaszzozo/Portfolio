# Tic Tac Toe

-PL- 🇵🇱
Ten projekt to gra w kółko i krzyżyk na ekranie konsoli z udziałem sztucznej inteligencji. AI korzysta tutaj z algorytmu MiniMax dodatkowo wzbogaconego o własną implementację alfa-beta cięć dostosowaną do stworzonej własnoręcznie klasy drzewa stanów.

-EN- 🇬🇧
This project is a tic-tac-toe game on the console screen involving artificial intelligence. AI uses the MiniMax algorithm here, additionally enriched with my own implementation of alpha-beta cuts adapted to the tree class created for this project.
